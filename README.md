# mkcmd
> 個人的によく行う処理をコマンド化していくリポジトリ

コマンド | 用途
--- | ---
[`node show-argument.js`](./src/show-arguments.ts) | コマンドライン引数を表示する


## 導入方法
### 初回
1. GitHub Release に添付されているZip ファイルをダウンロードする
1. ダウンロードしたファイルを任意の場所で解凍する
1. `.node-version` を確認し、必要なNode.js を整備する

### 更新
1. 前回のものを削除する
1. 「初回」と同様の手順を行う
